# Employee Attrition Prediction

## Overview

This project predicts whether an employee is likely to leave the company using a Machine Learning model built with Logistic Regression.

## Features

- Employee Information Form
- Job Details Input
- Work Experience Input
- Performance Details
- Attrition Prediction
- Stay & Attrition Probability
- Model Confidence Score
- Interactive Streamlit UI

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Streamlit
- Joblib

## Machine Learning Algorithm

- Logistic Regression

## Project Structure
